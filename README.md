<h1>Data Pipeline - Crypto API</h1>

Projeto de pipeline de dados utilizando Python para ingestão, transformação e armazenamento de dados de criptomoedas.

**Tecnologias:**

* Python

* Pandas

* API REST

**Pipeline:**

* Extração de dados via API CoinGecko

* Transformação com Pandas

* Armazenamento em CSV
